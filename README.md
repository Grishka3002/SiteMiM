# VVGU Ticketing MVP

Локальный MVP сайта для бронирования мест на мероприятие "Мисс и Мистер ВВГУ".

## Что уже есть

- визуальная схема зала с выбором мест;
- ввод ФИО и группы для каждого выбранного места;
- генерация билетов с уникальным кодом и QR;
- админ-панель со списком гостей и ручной сменой статуса;
- страница волонтёров для проверки билетов по QR-коду или вручную.

## Как запустить

Запустите сайт одной командой:

```powershell
npm start
```

После этого откройте `http://localhost:8080`.

## Ограничение MVP

Сейчас данные хранятся в `localStorage` браузера. Для реального использования на нескольких устройствах нужен backend, общая база данных и серверная генерация QR/билетов.
